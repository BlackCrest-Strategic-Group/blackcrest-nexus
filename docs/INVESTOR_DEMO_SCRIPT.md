# BlackCrest Investor Demo Script (5 Minutes)

1. **Open landing page**
   - Show positioning: role-based procurement intelligence OS, public/synthetic demo by default.
2. **Launch investor demo**
   - Click **Launch Investor Demo** and explain one-click synthetic data load.
3. **Show CEO view**
   - Highlight spend under review, leakage estimate, supplier risk exposure, open opportunity value.
4. **Show margin leak alerts**
   - Walk through top leakage alerts and recommended actions.
5. **Show RFP analysis report**
   - Generate RFP report with executive summary, bid/no-bid score, compliance checklist.
6. **Show blanket PO builder**
   - Upload sample spreadsheet and show normalized output / grouping flow.
7. **Show ERP connector center**
   - Demonstrate SAP/Oracle/Infor/Dynamics profile model, read-only defaults, IT approval posture.
8. **Show role-based buyer action list**
   - Switch between Buyer and Purchasing Assistant perspectives.
9. **Close with value proposition + expansion path**
   - “BlackCrest identifies leakage, reduces concentration risk, operationalizes sourcing decisions, and scales into customer-controlled ERP integrations.”
