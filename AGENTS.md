# BlackCrest Nexus AGENTS.md

## Product Position
BlackCrest Nexus is an AI Procurement Operations Platform designed to unify procurement intelligence, supplier analytics, sourcing workflows, operational visibility, and AI-assisted procurement analysis into a centralized enterprise-ready platform.

---

# Core Product Focus

## Primary Platform Goals
- Procurement intelligence
- Supplier analytics
- RFQ workflows
- Executive dashboards
- Operational visibility
- Regional supplier intelligence
- Enterprise procurement operations

---

# Workflow Priorities

## Primary Investor Demo Workflow
1. Upload RFQ or procurement package
2. AI analyzes uploaded files
3. Supplier risks surfaced
4. Procurement insights generated
5. Executive summary displayed
6. Dashboard metrics updated

Supported uploads:
- PDF
- XLSX
- DOCX

# Deployment Rules

Deployment requirements:
- Preserve Render compatibility
- Preserve Vite build stability
- Avoid breaking package.json
- Validate imports before changes
- Keep frontend and backend deployment-safe
- Maintain clean routing structure

"The operational intelligence layer for procurement teams."

The platform should focus on:
- Procurement operations
